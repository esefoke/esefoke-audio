# Esefoke Audio

Free AI Audio Narration Plugin for WordPress powered by Edge-TTS.

## Features

* Free AI voices
* Multiple voice selection
* Speech speed control
* Automatic audio generation
* Audio Library
* Download MP3
* Frontend audio player
* Generate Missing Audio
* Edge-TTS integration
* No API fees

## Requirements

* WordPress 6.0+
* PHP 8.0+
* Python 3.11 or later
* Edge-TTS

## Installation

1. Upload the plugin.
2. Activate it.
3. Install Edge-TTS.
4. Enter the Edge-TTS path.
5. Generate audio.

## Supported Voices

* Ryan (UK Male)
* Sonia (UK Female)
* Jenny (US Female)
